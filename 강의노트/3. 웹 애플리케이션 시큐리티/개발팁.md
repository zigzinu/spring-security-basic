# 개발팁

## 브라우져에서 서버에 요청한 내용

- 크롬 개발자 모드에서 네트워크 탭을 보면, `/` 주소만 요청이 들어왔는데 `/favicon.ico` `302` 와 `/login` `200` 까지 요청된 것을 확인할 수 있다.
- `/favicon.ico`에 대한 요청은 브라우져에서 보내는 것인데, 이 요청에 대한 처리가 거부되어 `/login` 까지 연결된다.
  - `.anyRequest().authenticated()`에 걸린다.
